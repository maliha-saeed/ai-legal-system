# AI Legal Intake System

AILegal is a Flask-based web application designed to support early-stage legal workflows.

## Features
- Structured client intake form
- Rule-based claim classification
- Viability screening
- Document upload and information extraction
- Template-based draft document generation

## Technologies
- Python
- Flask
- SQLite
- spaCy
- HTML/CSS/Jinja2

## How to Run
1. Create a virtual environment
2. Install requirements
3. Run `python3 app.py`
4. Open `http://127.0.0.1:5000`
